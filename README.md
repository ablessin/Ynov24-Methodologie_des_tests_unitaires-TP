### API Endpoints
#### POST /users
Create a new user.

##### Request Body
```
{
  "name": "John Doe",
  "email": "johndoe@example.com",
  "password": "password123"
}
```

##### Response Body
```
{
  "id": 1,
  "name": "John Doe",
  "email": "johndoe@example.com"
}
```

#### GET /users
Get a list of all users.

##### Response Body
```
[
  {
    "id": 1,
    "name": "John Doe",
    "email": "johndoe@example.com"
  },
  {
    "id": 2,
    "name": "Jane Smith",
    "email": "janesmith@example.com"
  }
]
```

#### GET /users/:id
Get one user by id.

##### Response Body
```
{
    "id": 1,
    "name": "John Doe",
    "email": "johndoe@example.com"
}
```
